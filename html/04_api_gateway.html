¿Qué hace el API Gateway?
🔀

Enrutamiento

Dirige cada request al microservicio correcto según la ruta y método HTTP.

🔐

Autenticación

Valida tokens JWT o API keys antes de que la petición llegue a los servicios.

🚦

Rate Limiting

Limita el número de requests por cliente para proteger los servicios del abuso.

⚡

Caché

Almacena respuestas frecuentes reduciendo la carga en los microservicios.
